Readme Neuner (Patience/Solitaire)
-------------------------------------------------
Neuner: (c) Ulrike Jahnke-Soltau (uja)

Dependencies/Voraussetzungen:
-----------------------------
Linux, Qt(3)

Installation:
-------------
Start  ./install_neuner_engl.cmd (english version)
Starte ./install_neuner_deu.cmd  (deutsche Version)

Spiel starten/start game:
cd neuner
./neuner
